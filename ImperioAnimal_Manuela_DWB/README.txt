
ImperioAnimal_Manuela_online - Proyecto entregable (frontend responsive, images from internet)
-----------------------------------------------------------------------------------------
Contenido:
- frontend/: HTML responsive with Bootstrap, CSS and JS, logo local and images loaded from Internet (Unsplash URLs).
- The pages are demo with simulated forms (alert).

Cómo ver el sitio (simple):
1. Extrae la carpeta ImperioAnimal_Manuela_online.zip.
2. Abre la carpeta 'frontend' y haz doble clic en index.html para abrirlo en el navegador.
   (Las imágenes cargan desde Internet; necesitas conexión para verlas.)
3. Si alguna imagen no carga por CORS al abrir localmente, usa Live Server en VS Code o python -m http.server 5500.

Notas:
- Las imágenes provienen de Unsplash (URLs) y se muestran directamente.
- Los iconos de redes son decorativos y no enlazan a sitios externos.
- Si quieres que descargue las imágenes y las deje locales (offline), dímelo y lo hago.
